Switch 判斷式

Switch 判斷式用來根據不同的條件執行不同的程式碼區塊，通常用於多重選擇的情況。

        switch (表達式) {
            case 值1:
                // 當表達式等於值1時執行的程式碼
                break;
            case 值2:
                // 當表達式等於值2時執行的程式碼
                break;
            ...
            default:
                // 當表達式不符合任何情況時執行的程式碼
        }

迴圈

迴圈用來重複執行一段程式碼，直到特定條件不再符合為止。

for 迴圈

        for (初始化計數器; 執行條件; 步進值) {
            // 重複執行的程式碼
        }

for/in

針對陣列的項目做處理

        for (變數 in 陣列物件) {
            // 重複執行的程式碼
        }

while 迴圈

        while (執行的條件) {
            // 重複執行的程式碼
        }

do while 迴圈

無論如何最少會跑一次

        do {
            // 重複執行的程式碼
        } while (執行的條件);

brack 與 continue

        // break 用來跳出迴圈
        for (let i = 0; i < 10; i++) {
            if (i == 5) {
                break; // 當 i 等於 5 時跳出迴圈
            }
            // 其他程式碼
        }

        // continue 用來跳過本次迴圈，進入下一次迴圈
        for (let i = 0; i < 10; i++) {
            if (i % 2 == 0) {
                continue; // 當 i 為偶數時，跳過本次迴圈
            }
            // 其他程式碼
        }
