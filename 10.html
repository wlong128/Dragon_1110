Bootstrap 5

斷點

  • xs(預設): 超小螢幕 (手機直向) 小於 576px
  • sm: 小螢幕 (手機橫向) 大於等於 576px
  • md: 中螢幕 (平板直向) 大於等於 768px
  • lg: 大螢幕 (平板橫向、小筆電) 大於等於 992px
  • xl: 超大螢幕 (一般筆電) 大於等於 1200px
  • xxl: 特大螢幕 (大型桌機) 大於等於 1400px

容器類

  • .container: 固定寬度容器，會隨斷點改變寬度
  • .container-fluid: 滿版寬度容器，寬度100%

間距

Bootstrap 提供了簡單的間距工具類，可以快速設置元素的外邊距（margin）和內邊距（padding）。

  • m - margin 外邊距
  • p - padding 內邊距
  • t - top 上方
  • b - bottom 下方
  • s - start 左方 (LTR 文字方向)
  • e - end 右方 (LTR 文字方向)
  • x - 左右
  • y - 上下
  • 數值範圍: 0, 1, 2, 3, 4, 5, auto (0 為無間距，1-5 為不同大小的間距，auto 為自動)
內邊距 p-3，外邊距 mt-3
內邊距 p-2，外邊距 mt-2
內邊距 p-5，外邊距 mt-5

標題

Bootstrap 提供了預設的標題樣式，可以使用 h1 到 h6 標籤來表示不同層級的標題。

Display

文字大小可使用 display 類別來設置，範圍從 display-1 (最大) 到 display-6 (最小)。

烏龍院

Color

Bootstrap 提供了多種文字顏色類別，可以快速設置文字顏色。

  • primary - 重要的色彩
  • secondary - 次要的色彩
  • success - 成功的色彩
  • danger - 危險的色彩
  • warning - 警告的色彩
  • info - 資訊的色彩
  • light - 淺色系
  • dark - 深色系
  • muted - 次要文字色彩

圖片

使用 .img-fluid 類別可以讓圖片在不同裝置上自動調整大小，保持響應式設計。

可先安裝「Simple Placeholder Images」外掛，即可輸入「place」即時產生隨機圖片

可使用 .rounded 類別來設置圖片的圓角效果，.rounded-circle 可將圖片設置為圓形，.rounded-5 可設置較大的圓角。
