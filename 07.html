自訂函數

函數是用來封裝一段可以重複使用的程式碼區塊，可以用來執行特定的任務或計算。

在 Javascript 中，可以使用 function 關鍵字來宣告一個函數。

        // 副程式
        function 函數名稱(參數) {
            // 函數內的程式碼
            // 不需要回傳
        }

結果將顯示在此

有回傳值的函數

有回傳值的函數可以使用 return 關鍵字來回傳一個值，這個值可以被呼叫函數的地方接收並使用。

        // 有回傳值的函數
        function 函數名稱(參數) {
            // 函數內的程式碼
            return 回傳值;
        }

MVC

MVC 是一種軟體設計模式，用來將應用程式分成三個主要部分： 模型 (Model)、視圖 (View) 和控制器 (Controller)。

變數與常數

變數是用來儲存資料的容器，可以用來存放數字、文字、布林值等各種資料型態。

在 Javascript 中，可以使用 var、let 或 const 關鍵字來宣告變數。

MSG

物件

物件是用來表示具有屬性和方法的實體，可以用來封裝相關的資料和行為。

        var car = {
            color: "red",
            model: "Toyota",
            year: 2020,
            start: function() {
                return "This car is starting.";
            },
            stop: function() {
                return "This car is stopping.";
            }
        }

Array Message

判斷式

判斷式是用來根據條件執行不同程式碼區塊的結構，可以用來實現分支邏輯。

IF 判斷

        if (條件) {
            // 條件為真時執行的程式碼
        } else {
            // 條件為假時執行的程式碼
        }

多條件 IF 判斷

        if (條件1) {
            // 條件1 為真時執行的程式碼
        } else if (條件2) {
            // 條件2 為真時執行的程式碼
        } else {
            // 以上條件皆不為真時執行的程式碼
        }
